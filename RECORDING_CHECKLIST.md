# Recording Checklist — Leer Afrikaans

Save each file as **`public/audio/{id}.mp3`** (e.g. `public/audio/g1.mp3`).  
The app picks them up automatically — no code changes needed.

**Tips for recording:**
- Quiet room, no background noise
- Speak at a natural, slightly slower-than-normal pace
- Keep each recording to just the phrase — no intro/outro
- Aim for consistent volume across all files
- Any format works for recording; just export/convert to MP3 before saving
